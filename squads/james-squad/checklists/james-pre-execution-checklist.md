# James Pre-Execution Checklist

Execute ANTES de qualquer task de qualquer squad.

## 1. Identificação
- [ ] Task identificada com nome claro
- [ ] Squad de origem identificado
- [ ] Agente solicitante identificado
- [ ] Contexto/justificativa fornecido

## 2. Alinhamento Estratégico
- [ ] Task está alinhada com objetivos do projeto atual
- [ ] Task está dentro do escopo da story/epic ativa
- [ ] Não há tasks de maior prioridade bloqueadas esperando

## 3. Dependências
- [ ] Todas as tasks pré-requisito foram concluídas
- [ ] Não há bloqueios externos pendentes
- [ ] Sequenciamento está correto

## 4. Autoridade do Agente (agent-authority.md)
- [ ] O agente solicitante tem autoridade para esta task
- [ ] Se operação exclusiva de outro agente → REDIRECT

## 5. Risco
- [ ] Risco estimado: low | medium | high
- [ ] Se high → justificativa detalhada presente
- [ ] Impacto no projeto avaliado

## 6. Decisão
- [ ] Decisão documentada: APPROVED | REJECTED | REDIRECTED | PENDING
- [ ] Justificativa registrada
- [ ] Agente solicitante comunicado
