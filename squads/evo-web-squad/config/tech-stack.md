# Tech Stack — evo-web-squad

## Opções de Stack

| Stack | Quando Usar |
|-------|------------|
| HTML/CSS/JS puro | Landing pages de alta performance, máxima velocidade |
| Astro | Sites estáticos com componentes, blog, documentação |
| Next.js | Sites com SSR, e-commerce, autenticação |
| WordPress | Clientes que precisam editar conteúdo sem código |
| Webflow | Projetos design-driven sem time técnico |

## Analytics & Tracking
- Google Analytics 4 (GA4)
- Google Tag Manager (GTM)
- Meta Pixel
- Google Search Console

## Performance
- Cloudflare (CDN + DNS)
- ImageKit ou Cloudinary (otimização de imagens)
- Lighthouse CI

## Hospedagem Preferencial
- Vercel (Next.js, Astro)
- Netlify (sites estáticos)
- WP Engine (WordPress)
- GitHub Pages (landing pages simples)
